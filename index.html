React Simple Date Picker

Simple Date Picker component for React

View on GitHub

Try It Out

Installation

The easiest way to use react-simple-datepicker is to install it from npm and include it in your React build process.

  npm install -save react-simple-datepicker

Example

    var React = require('react');
    var DatePicker = require('react-simple-datepicker');
    require('react-simple-datepicker/dist/index.css');

    var DatePickerDemo = React.createClass({
      getInitialState: function () {
        return {
          date: new Date()
        }
      },

      render: function () {
        return <DatePicker date={this.state.date}>;
      }
    });

    React.render(<DatePickerDemo />, document.body);
